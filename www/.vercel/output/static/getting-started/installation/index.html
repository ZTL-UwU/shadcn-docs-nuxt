Installation

How to install shadcn-docs-nuxt in your app.

Using the starter template

Get Starter Template

npm
pnpm
bun
yarn
npx nuxi@latest init -t github:ZTL-UwU/shadcn-docs-nuxt-starter

pnpm dlx nuxi@latest init -t github:ZTL-UwU/shadcn-docs-nuxt-starter

bun x nuxi@latest init -t github:ZTL-UwU/shadcn-docs-nuxt-starter

yarn dlx nuxi@latest init -t github:ZTL-UwU/shadcn-docs-nuxt-starter

Alternatively, you can clone or download the template from the GitHub repo.

Navigating to the Project Directory

cd [project-name]

Development Server

npm
pnpm
bun
yarn
npm run dev

pnpm run dev

bun run dev

yarn run dev

Well done! A browser window should automatically open for http://localhost:3000.
