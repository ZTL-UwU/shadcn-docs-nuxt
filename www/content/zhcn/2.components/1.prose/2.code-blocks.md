### Transformers

::alert{type="info"}
移除标记中的 \ 以启用 transformers。

例如: `// [\!code --]` to `// [!code --]`
::

#### 差异

::stack
::div{class="p-4"}
```ts
console.log('hewwo'); // [!code --]
console.log('hello'); // [!code ++]
console.log('goodbye');
```
::
````md
```ts
console.log('hewwo'); // [\!code --]
console.log('hello'); // [\!code ++]
console.log('goodbye');
```
````
::

::stack
