### 突出显示图像

突出显示图像是一种带有边框和阴影的图像, 使其更加突出.

::stack
  ::div{class="p-4"}
  ![Lifted Image](/prose-img/light.png){lifted}
  ::

  ```md
  ![Lifted Image](/prose-img/light.png){lifted}
  ```
::

### 颜色模式图像

颜色模式图像是指根据当前颜色模式 (浅色或深色) 而变化的图像. 这对于在浅色和深色模式下颜色不同的图像非常有用.

::stack
  ::div{class="p-4"}
    ::color-mode-image
    ---
    srcLight: /prose-img/light.png
    srcDark: /prose-img/dark.png
    alt: Color Mode Image
    ---
    ::
  ::

  ```mdc
  ::color-mode-image
  ---
  srcLight: /prose-img/light.png
  srcDark: /prose-img/dark.png
  alt: Color Mode Image
  ---
  ::
  ```
::

### 图像缩放

点击图片可以放大查看. 此功能默认启用. 你可以在 app config 中将 `main.imageZoom` 设置为 `false` 来全局禁用缩放, 或者在图片本身上设置 `:zoom="false"` 来禁用单独图片的缩放.

::stack
  ::div{class="p-4"}
  ![No Zooming Image](/prose-img/light.png){:zoom="false"}
  ::

  ```md
  ![No Zooming Image](/prose-img/light.png){:zoom="false"}
  ```
::
