## 动机

shadcn-docs-nuxt 作为 Docus 和 Nuxt UI Pro Docs 的免费替代文档解决方案而创建.

## 特征

- 免费且[开源](https://github.com/ZTL-UwU/shadcn-docs-nuxt).
- 完全[可定制](/api/configuration).
- 丰富的[组件](/components/prose)可供使用.
- 移动端支持.
- 由 Nuxt Content 提供支持的索引搜索.
- 与 `Docus`, `Nuxt UI Pro Docs`, `undocs` 部分组件兼容.

## 鸣谢

- [Nuxt Content](https://content.nuxt.com/): 为 Vue 开发者轻松制作内容.
- [shadcn-ui](https://ui.shadcn.com/): 用于精美的组件设计与文档设计.
- [shadcn-vue](https://www.shadcn-vue.com/): 用于 shadcn-ui 的 Vue 移植及一些文档组件源码.
- [Docus](https://docus.dev/): 用于灵感和一些文档组件源码.
- [Nuxt UI Pro Docs](https://docs-template.nuxt.dev/): 用于灵感.
