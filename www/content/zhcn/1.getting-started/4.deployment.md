## 静态托管

::steps{:level="4"}
#### 在 `nuxt.config.ts` 或 `.env` 中配置 `site.url`

OG 图片必须使用绝对 URL 提供, 您需要提供站点 URL.

::code-group
```ts [nuxt.config.ts]
export default defineNuxtConfig({
  site: {
    url: 'https://example.com',
  },
});
```
```dotenv [.env]
NUXT_SITE_URL=https://example.com
```
::

#### 运行生成
:pm-run{script="generate"}

::alert{icon="lucide:triangle"}
在 Vercel 上, 将**构建命令**设置为 `nuxt generate`, 位于**构建与开发设置**中.
::

#### 部署
您现在可以将 `.output/public` 目录部署到任何静态托管服务, 或使用 `npx serve .output/public` 在本地预览.
::

## SSR 服务器

::steps{:level="4"}
#### 运行构建
:pm-run{script="build"}

#### 部署
```shell
node .output/server/index.mjs
```
::
