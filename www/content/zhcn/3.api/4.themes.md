## UI 主题

```ts [app.config.ts]
export default defineAppConfig({
  shadcnDocs: {
    theme: {
      customizable: true,
      color: 'zinc',
      radius: 0.5,
    },
  },
});
```

::field-group
  ::field{name="customizable" type="boolean" defaultValue="true"}
  主题是否可以由用户自定义 (在标题中显示自定义按钮).
  ::
  ::field{name="color" type="Color" defaultValue="zinc"}
  默认的颜色主题. https://www.shadcn-vue.com/themes.html
  ::
  ::field{name="radius" type="string" defaultValue="0.5"}
  默认半径.
  ::
::

::Alert{to="/api/configuration/shadcn-docs#theme"}
请参阅 **配置** 中的 `theme`.
::

## 使用其它字体

你可以在 `tailwind.config.js` 文件中自定义你的字体. _shadcn-docs-nuxt_ 在底层使用 [`@nuxt/fonts`](https://fonts.nuxt.com/), 所以你可以使用任何你想要的字体. 默认字体是 [Geist Font](https://vercel.com/font).

```js [tailwind.config.js]
export default {
  theme: {
    extend: {
      fontFamily: {
        sans: ['Geist'],
        mono: ['Geist Mono'],
      },
    },
  },
};
```
