## бһҖбһ¶бһҡбһӣбһҫбһҖбһ‘бһ№бһҖбһ…бһ·бһҸбҹ’бһҸ

**shadcn-docs-nuxt** бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһҳбһҖбһҖбҹ’бһ“бһ»бһ„бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҮбҹҶбһ“бһҪбһҹбһІбҹ’бһҷбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ [Docus](https://docus.dev/) бһ“бһ·бһ„ [Nuxt UI Pro Docs](https://docs-template.nuxt.dev/)бҹ”

## бһҳбһ»бһҒбһ„бһ¶бһҡ

- бһҘбһҸбһӮбһ·бһҸбһҗбҹ’бһӣбҹғ бһ“бһ·бһ„ [бһ”бһҫбһҖбһ”бҹ’бһҡбһ—бһ–бһҖбһјбһҠ](https://github.com/ZTL-UwU/shadcn-docs-nuxt)бҹ”
- бһўбһ¶бһ… [бһҖбҹӮбһ…бҹ’бһ“бҹғбһҸбһ¶бһҳбһ…бҹҶбһҺбһ„бҹӢбһ…бҹҶбһҺбһјбһӣбһ…бһ·бһҸбҹ’бһҸ](/api/configuration)бҹ”
- бһҹбһҳбҹ’бһ”бһјбһҡбһ‘бҹ…бһҠбҹ„бһҷ [components](/components/prose) бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бҹ”
- бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһӣбһҫбһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘бҹ”
- бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҠбҹ„бһҷ Search Engine бһҠбҹӮбһӣбһҳбһ¶бһ“бһҮбҹҶбһ“бһҪбһҷбһ–бһё Nuxt Contentбҹ”
- бһҳбһ¶бһ“бһ”бһ“бҹ’бһҗбҹӮбһҳ component бһҠбҹӮбһӣбһҳбһҖбһ–бһё `Docus`, `Nuxt UI Pro Docs`, `undocs`бҹ”

## бһҖбҹ’бһҡбҹҒбһҢбһёбһҸ

- [Nuxt Content](https://content.nuxt.com/): бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ¶бһҸбһ·бһҖбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Vue Developersбҹ”
- [shadcn-ui](https://ui.shadcn.com/): бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ component бһҠбҹӮбһӣбһҹбҹ’бһҡбһҹбҹӢбһҹбҹ’бһўбһ¶бһҸ бһ“бһ·бһ„бһҹбһҳбҹ’бһ”бһјбһҡбһ”бҹӮбһ”бҹ”
- [shadcn-vue](https://www.shadcn-vue.com/): бһҮбһ¶бһҖбһ¶бһҡбһ“бһ¶бҹҶбһ…бһјбһӣ component бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Vue бһ–бһё shadcn-ui бһ“бһ·бһ„бһ”бҹ’бһҡбһ—бһ–бһ•бҹ’бһҹбҹҒбһ„бҹ—бһ‘бҹҖбһҸбҹ”
- [Docus](https://docus.dev/): бһҷбһҖбһҮбһ¶бһӮбһҳбҹ’бһҡбһј бһ“бһ·бһ„бһҖбһ¶бһҡбһҡбһ…бһ“бһ¶ component бһҳбһҪбһҷбһ…бҹҶбһ“бһҪбһ“бҹ”
- [Nuxt UI Pro Docs](https://docs-template.nuxt.dev/): бһҷбһҖбһҮбһ¶бһӮбһҳбҹ’бһҡбһјбҹ”
