## ការលើកទឹកចិត្ត

**shadcn-docs-nuxt** ត្រូវបានបង្កើតឡើងមកក្នុងគោលបំណងជំនួសឲ្យការប្រើប្រាស់ [Docus](https://docus.dev/) និង [Nuxt UI Pro Docs](https://docs-template.nuxt.dev/)។

## មុខងារ

- ឥតគិតថ្លៃ និង [បើកប្រភពកូដ](https://github.com/ZTL-UwU/shadcn-docs-nuxt)។
- អាច [កែច្នៃតាមចំណង់ចំណូលចិត្ត](/api/configuration)។
- សម្បូរទៅដោយ [components](/components/prose) ជាច្រើន។
- គាំទ្រលើការបង្ហាញទូរស័ព្ទ។
- ងាយស្រួលត្រូវបានស្វែងរកដោយ Search Engine ដែលមានជំនួយពី Nuxt Content។
- មានបន្ថែម component ដែលមកពី `Docus`, `Nuxt UI Pro Docs`, `undocs`។

## ក្រេឌីត

- [Nuxt Content](https://content.nuxt.com/): ងាយស្រួលបង្កើតមាតិកាសម្រាប់ Vue Developers។
- [shadcn-ui](https://ui.shadcn.com/): សម្រាប់ component ដែលស្រស់ស្អាត និងសម្បូរបែប។
- [shadcn-vue](https://www.shadcn-vue.com/): ជាការនាំចូល component សម្រាប់ Vue ពី shadcn-ui និងប្រភពផ្សេងៗទៀត។
- [Docus](https://docus.dev/): យកជាគម្រូ និងការរចនា component មួយចំនួន។
- [Nuxt UI Pro Docs](https://docs-template.nuxt.dev/): យកជាគម្រូ។
