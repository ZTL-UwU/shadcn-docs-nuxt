### `useConfig()`

从 `app.config.ts` 获取配置。

- `main`、`header`、`aside`、`footer` 和 `toc` 可以在 `_dir.yml` 和任何 `page.md` 文件中设置。

```ts
const {
  site,
  search,
  header,
  main,
  aside,
  footer,
  toc,
} = useConfig();
```
